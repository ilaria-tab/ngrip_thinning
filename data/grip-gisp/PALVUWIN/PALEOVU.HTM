There are three methods for running PaleoVu on Windows 3.1.x/95/NT:
(1) Launching PaleoVu directly from this page:
  To launch paleovu directly from this web page, you must first set a MIME type, and then Launch PaleoVu.
    Set the MIME type
Launch PaleoVu
  NOTE: Setting a MIME type is a somewhat complicated process. If you experience difficulty setting the MIME type try one of the other two methods for launching PaleoVu described below.
     
(2) Launching PaleoVu from this CD-ROM:
  "PaleoVu.exe" is located in the palvuwin directory of this CD-ROM. For example, if your CD-ROM drive is the drive is E, PaleoVu is located in E:\palvuwin\paleovu.exe. Use the Run dialog (Windows 3.x/95/NT), the File Manager (Windows 3.x) , or Windows Explorer (Windows 95/NT) to launch PaleoVu.
     
(3) Installing PaleoVu on your hard drive:
  For better performance, PaleoVu can be installed and run from your hard drive.
     
  For Windows 95 and Windows NT:
    Copy the palvuwin directory of the CD-ROM to the top level of a drive (such as c:\ or d:\) of your PC. In Windows Explorer drag the palvuwin directory from the CD-ROM drive to the top-level of a logical drive letter of your PC. NOTE: palvuwin must be a top-level directory.
     
    The PaleoVu application icon is located in the palvuwin directory and can be launched by double clicking on the icon in Windows Explorer, or run from the Start menu.
     
  For Windows 3.1.x:
    Copy the palvuwin directory of the CD-ROM to the top-level of a drive (such as c:\ or d:\) of your PC. In File Manager drag the palvuwin folder from the CD-ROM drive onto a drive letter of your PC. NOTE: the palvuwin directory must be a top-level directory.
     
    The PaleoVu application icon is located in the palvuwin directory and can be launched by double-clicking on the icon in the File Manager, or by using the Run dialog located in the Program Manager's File menu.
     
Hardware/Software Requirements
  Minimum System Configuration For MS Windows:
   
  • An 80486 CPU Configuration, with 8MB of RAM.
  • A VGA monitor.
  • 2x CD-ROM Drive
  • Windows 3.1, running in 386 enhanced mode with a compatible mouse.
  • If you install PaleoVu to your hard drive, 35MB of hard disk space is required.
Please refer to the PaleoVu Users' Guide for documentation.
